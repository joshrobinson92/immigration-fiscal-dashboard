# Immigrant Fiscal Impact Dashboard

An interactive, academic-viewpoint dashboard exploring the net fiscal impact of immigration on the U.S. economy, built using data from the Cato Institute's 2026 study: *"Immigrants' Recent Effects on Government Budgets: 1994–2023"*.

The live application is hosted via GitHub Pages at:
👉 **[joshrobinson92.github.io/immigration-fiscal-dashboard](https://joshrobinson92.github.io/immigration-fiscal-dashboard/)**

---

## 📊 Features

*   **Real-time Recalculations**: Adjust policy sliders and toggles in the sidebar to see immediate updates in per-capita and total fiscal impact.
*   **Immigrant Status Categorization**: Compare contributions across different categories: **Naturalized Citizens**, **Visa / Temporary (Legal Noncitizens)**, and **Undocumented (Illegal)**.
*   **Timeframe Toggle**: Switch between **Annualized Average** (default) and **30-Year Cumulative** values.
*   **Cost Selection Buckets**: Choose which government outlays (K-12 Education, Welfare/Needs-based, Old-Age Benefits, Congestible Goods, Incarceration, and Higher Education) are factored into the net contribution.
*   **Interactive Visualizations**: Multiple Chart.js visual panels displaying:
    *   Taxes Paid vs. Government Spending.
    *   Per-Capita Net Footprints.
    *   Education & Demographic Cost/Tax Breakdowns.
*   **Raw Data Explorer**: A built-in JSON viewer that allows you to inspect, copy, or download the parsed raw data as JSON or CSV.

---

## 🛠️ Policy Controls (Addressing Academic Criticisms)

The dashboard integrates interactive controls modeling major academic and political parameters:
1.  **Fixed Costs / Public Goods (Defense & Past Debt)**: Toggles between Cato's marginal cost model ($0 defense/interest cost allocated to immigrants) and NASEM's average cost model (pro-rata distribution based on population share, with a percentage slider).
2.  **U.S.-Born Child Attribution**: Toggles between Cato's model (no attribution), CIS's model (attributing child education/health costs to parents while excluding second-generation adult earnings), and a Full Multigenerational approach.
3.  **Undocumented Tax Compliance**: A slider to adjust undocumented income/payroll tax compliance from 0% to 100% (baseline assumes 60%).
4.  **Indirect Property Taxes**: Toggle to credit immigrants with indirect property taxes generated by U.S.-born homeowners due to housing demand.

## 🚀 Running Locally

Since this dashboard is a fully static client-side web application, you do not need to install complex dependencies. 

Simply run a local HTTP server in the root folder:

```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve .
```

Then open **`http://localhost:8000`** in your browser.
